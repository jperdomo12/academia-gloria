Este documento define la evolución funcional de la Academia de Gloria.

El Roadmap describe los grandes hitos del proyecto y sirve como guía para su crecimiento a medio y largo plazo.

No documenta tareas individuales, sino las capacidades que irán incorporándose a la plataforma en cada versión.

## 🚧 Versión 2.0 – Plataforma Cloud

### Objetivo

### Incluye

- Firebase
- Authentication
- Cloud Firestore
- API Academia
- Calendario Escolar

- Sincronización Cloud

### Estado

✅ Finalizada
