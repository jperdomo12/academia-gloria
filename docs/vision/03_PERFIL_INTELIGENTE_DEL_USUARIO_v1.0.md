# Propósito

La Academia no debe conocer únicamente un nombre de usuario.

Debe conocer a la persona para poder acompañarla, motivarla y adaptar la
experiencia de aprendizaje a sus necesidades.

> **La Academia no adapta al niño a la tecnología.**
>
> **La Academia adapta la tecnología a cada niño.**

------------------------------------------------------------------------

# Visión

Cada usuario tendrá un perfil vivo que irá creciendo con el tiempo.

No será una ficha administrativa. Será una representación de cómo
aprende, qué le gusta, cuáles son sus fortalezas y cómo podemos ayudarle
a seguir creciendo.

# Perfil básico

-   Nombre
-   Fecha de nacimiento
-   Avatar
-   Idioma
-   Zona horaria
-   Colegio
-   Curso
-   Curso escolar

# Preferencias

-   Color favorito
-   Animales favoritos
-   Historias favoritas
-   Personajes favoritos
-   Juegos favoritos
-   Comidas favoritas
-   Música favorita

# Perfil educativo

## Lectura

-   Nivel
-   Velocidad
-   Comprensión
-   Fluidez
-   Pronunciación

## Escritura

-   Ortografía
-   Expresión
-   Creatividad

## Matemáticas

-   Cálculo
-   Razonamiento
-   Resolución de problemas

# Perfil emocional

Ejemplos:

-   Me gusta ayudar.
-   Me cuesta hablar delante de otras personas.
-   Soy perseverante.
-   Me pongo nervioso al leer.
-   Me encanta imaginar historias.

# Objetivos personales

-   Leer 15 minutos diarios.
-   Hablar con más confianza.
-   Contar un cuento completo.
-   Hacer una nueva amiga.
-   Participar más en clase.

# Logros

-   ⭐ Primera historia leída.
-   ⭐ Primera grabación.
-   ⭐ Cinco días seguidos leyendo.
-   ⭐ Primera presentación en clase.

# Lo que aprende la Academia

Con el tiempo podrá descubrir patrones como:

-   disfruta historias de animales;
-   mejora cuando escucha primero el cuento;
-   responde mejor por las mañanas;
-   lee con mayor tranquilidad cuando no tiene prisa.

# Personalización

Ejemplo:

> Hola Gloria 🌈.

> Hoy preparé una historia sobre una pequeña estrella porque sé que te
> gustan los cuentos donde los personajes descubren que pueden hacer
> cosas grandes.

La Academia dejará de ser una aplicación y se convertirá en una
compañera de aprendizaje.

# Evolución del avatar

-   Emoji
-   Avatar ilustrado
-   Avatar generado por IA

# Principios

1.  Cada niño es único.
2.  Nunca comparar niños.
3.  Celebrar el esfuerzo.
4.  Aprender jugando.
5.  La motivación es tan importante como el conocimiento.
6.  La tecnología debe adaptarse al niño.

# Aplicación a Gloria

Este diseño nace inspirado por Gloria, su familia, la logopeda, la
psicóloga infantil Azucena y la colaboración con el colegio.

Aunque la Academia nació para ayudar a Gloria, se diseña para que
cualquier niño pueda sentirse acompañado, comprendido y motivado.
