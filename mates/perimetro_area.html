IDEA CLAVE

Borde ≠ Dentro

📏 Perímetro

Es todo el borde de una figura.

🟩 Área

Es la superficie interior.

🧠 Truco

Perímetro → sumo lados. Área → uso fórmula.

📏 Perímetro = Borde

El perímetro es la longitud total alrededor de una figura.

Perímetro = sumar todos los lados

🟩 Área = Dentro

El área mide la superficie interior.

Área = superficie interior

Cuadrado

P = 4 × lado
A = lado × lado

Rectángulo

P = 2 × (b + h)
A = base × altura

Triángulo

P = a + b + c
A = (b × h) ÷ 2
